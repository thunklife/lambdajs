Module: numbers

numbers

Lambda functions that can be used to express natural numbers

Natural numbers can be expressed a number of successions from zero, and zero can be expressed as the identity function. For example, give a function 'zero' and a function 'succ' we can create the numbers 1-5:

var one    = succ(zero);
var two    = succ(succ(zero));
var three  = succ(succ(succ(zero)));
var four   = succ(succ(succ(succ(zero))));
var five   = succ(succ(succ(succ(succ(zero))))))

A succession is expressed as a pair, where the second item is the current predecessor; multiple successionis build up nested pairs.

This module depends on the foundation module.

NOTE

All functions accept a single parameter. Any functions that are documented as having multiple parameters will return a new function after the initial application, and so on until all arguments are supplied.

For example applySelf(func)(arg);

<static> iszero(n) → {function}

Given a number 'n', returns True if that number is zero, otherwise returns False.

Since a number can be expressed as a pair, where the first item is second, which is the same as untruth or False, we can select the first value from the pair to determine if the number is zero. When first is applied to a number, untruth is returned. When first is applied to zero which is identity, it returns itself (which is the same as truth or True)

Lambda notation: λ.n(n first);

<static> pred(n) → {function}

Given a number a number 'n', represented as a pair, return zero if the number is zero, otherwise return the second item from the pair.

Keeping in mind that a number is a pair, which is a function waiting for a selector, consider a number in the form: [false, [false, zero]]. Applying iszero returns untruth or False, which is the selector function second. second is then passed to 'n' to retrieve the second value. module:numbers.zero, and the previously selected value (the predecessor) are then passed to the result of iszero, which returns the predecessor.

For example:

### Finding the predecessor of 2
var two = succ(succ(zero)); //=> [untruth, [untruth, zero]
var selector = iszero(two); //=> untruth = second
var predecessor = selector(zero)(two(second)) //=> [false, zero] = one

### Finding the predecessor of 0
var selector = iszero(zero) //=> first. iszero is defined as n(first); therefore, if n is zero, which is the identity function, first is returned.
var predecessor = selector(zero)(zero(second)) //=> zero

Lambda notation: λ.n(((iszero n) zero) (n second))

<static> succ(n, s) → {function}

Given a number 'n', create a pair where second is the first item and 'n' is the second. In this way we can express numbers as such:

var one = succ(zero) //=> [second, zero];
var two = succ(one) //=> [second, [second, zero]]

Lambda notation: λn.λs((second) n)
